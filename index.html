Common CSS Properties

CSS JavaScript Description
background background sets all background style properties at once, such as color, image, origin and size, or repeat method
background-attachment backgroundAttachment sets whether a background image's position is fixed within the viewport, or scrolls with its containing block
background-color backgroundColor sets the background color of an element
background-image backgroundImage sets one or more background images on an element
background-position backgroundPosition sets the initial position for each background image relative to the position layer set by background-origin
background-repeat backgroundRepeat sets how background images are repeated
border border sets an element's border
border-bottom borderBottom sets an element's bottom border; sets the values of border-bottom-width, border-bottom-style and border-bottom-color
border-bottom-color borderBottomColor sets the color of an element's bottom border
border-bottom-style borderBottomStyle sets the line style of an element's bottom border
border-bottom-width borderBottomWidth sets the width of the bottom border of an element
border-color borderColor sets the color of an element's border
border-left borderLeft sets an element's left border
border-left-color borderLeftColor sets the color of an element's left border
border-left-style borderLeftStyle sets the line style of an element's left border
border-left-width borderLeftWidth sets the width of the left border of an element
border-right borderRight sets the properties of an element's right border
border-right-color borderRightColor sets the color of an element's right border
border-right-style borderRightStyle sets the line style of an element's right border
border-right-width borderRightWidth sets the width of the right border of an element
border-style borderStyle sets the line style for all four sides of an element's border
border-top borderTop sets all the properties of an element's top border
border-top-color borderTopColor sets the color of an element's top border
border-top-style borderTopStyle sets the line style of an element's top border
border-top-width borderTopWidth sets the width of the top border of an element
border-width borderWidth sets the width of an element's border
clear clear sets whether an element must be moved below (cleared) floating elements that precede it; applies to floating and non-floating elements
clip clip defines a visible portion of an element; applies only to absolutely positioned elements — that is, elements with position:absolute or position:fixed
color color sets the foreground color value of an element's text and text decorations, and sets the currentcolor value
cursor cursor sets the type of mouse cursor, if any, to show when the mouse pointer is over an element
display display sets whether an element is treated as a block or inline element and the layout used for its children, such as flow layout, grid or flex
filter filter applies graphical effects like blur or color shift to an element
float cssFloat places an element on the left or right side of its container, allowing text and inline elements to wrap around it
font font sets all the different properties of an element's font
font-family fontFamily specifies a prioritized list of one or more font family names and/or generic family names for the selected element
font-size fontSize sets the size of the font
font-variant fontVariant allows you to set all the font variants for a font
font-weight fontWeight sets the weight (or boldness) of the font
height height specifies the height of an element
left left participates in specifying the horizontal position of a positioned element
letter-spacing letterSpacing sets the horizontal spacing behavior between text characters
line-height lineHeight sets the height of a line box
list-style listStyle allows you set all the list style properties at once
list-style-image listStyleImage sets an image to be used as the list item marker
list-style-position listStylePosition sets the position of the ::marker relative to a list item
list-style-type listStyleType sets the marker (such as a disc, character, or custom counter style) of a list item element
margin margin sets the margin area on all four sides of an element
margin-bottom marginBottom sets the margin area on the bottom of an element
margin-left marginLeft sets the margin area on the left side of an element
margin-right marginRight sets the margin area on the right side of an element
margin-top marginTop sets the margin area on the top of an element
overflow overflow sets the desired behavior for an element's overflow — i.e. when an element's content is too big to fit in its block formatting context — in both directions
padding padding sets the padding area on all four sides of an element at once
padding-bottom paddingBottom sets the height of the padding area on the bottom of an element
padding-left paddingLeft sets the width of the padding area to the left of an element
padding-right paddingRight sets the width of the padding area on the right of an element
padding-top paddingTop sets the height of the padding area on the top of an element
page-break-after pageBreakAfter adjusts page breaks after the current element
page-break-before pageBreakBefore adjusts page breaks before the current element
position position sets how an element is positioned in a document
stroke-dasharray strokeDasharray defines the pattern of dashes and gaps used to paint the outline of a shape
stroke-dashoffset strokeDashoffset defines an offset on the rendering of the associated dash array
stroke-width strokeWidth defines the width of the stroke to be applied to a shape
text-align textAlign sets the horizontal alignment of a block element or table-cell box
text-decoration textDecoration sets the appearance of decorative lines on text
text-indent textIndent sets the length of empty space (indentation) that is put before lines of text in a block
text-transform textTransform specifies how to capitalize an element's text
top top participates in specifying the vertical position of a positioned element
vertical-align verticalAlign sets vertical alignment of an inline, inline-block or table-cell box
visibility visibility shows or hides an element without changing the layout of a document; can also hide rows or columns in a table
width width sets the width of the content area, but if box-sizing is set to border-box, it sets the width of the border area
z-index zIndex sets the z-order of a positioned element and its descendants or flex items
